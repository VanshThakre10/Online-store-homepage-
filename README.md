# SwiftCart - Modern E-commerce Store

Welcome to SwiftCart, a fully responsive, modern, and interactive e-commerce store frontend. 

## Project Overview

This project was built following a structured 10-step approach to guarantee a solid architecture and engaging user experience:

1. **Setup**: Foundational HTML and Tailwind configuration.
2. **Navbar**: A sticky, intuitive navigation bar with search and cart interactions.
3. **Hero Section**: A dynamic and visually appealing hero area to drive conversions.
4. **Categories**: Clear and structured product categories.
5. **Featured Products**: A showcase of top products with interactive hover states.
6. **Promotional Banner**: High-conversion newsletter and discount section.
7. **Testimonials**: Social proof to build trust with customers.
8. **Footer**: Comprehensive site links and company information.
9. **JavaScript Interactions**: Dynamic functionalities including add-to-cart feedback, dynamic search, mobile menus, and wishlist toggling.
10. **Final Deployment & Polish**: Project finalization, documentation, and remote repository push.

## Features

- Fully responsive design using Tailwind CSS.
- Modern aesthetics using Google Fonts (Inter) and custom glassmorphism effects.
- Interactive cart and wishlist counters.
- Clean and maintainable codebase.

## How to Run

Simply open `index.html` in your favorite browser. No build steps required.
